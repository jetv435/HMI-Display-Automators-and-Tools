HMI Display Head Automation - Python Bitmap Automator

Necessary Folder Structure:

..
- FunctionPackages
- bitmaps_IN
- ouput
- - bitmaps_OUT

Instructions:

1) Place bitmap .c files inside the 'bitmaps_IN' folder
2) Run 'Main.py'
3) Fill in the name of the last bitmap listed in the header file when prompted
4) Copy and paste relevant code from the .txt files in 'output' into the .h header file
5) Copy the new bitmap .c files from 'output -> bitmaps_OUT' into their proper locations
6) ALPHA NOTE: Some small corrections will likely be needed in the .h file
